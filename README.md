# Life Sync

**Life Sync** is a lightweight desktop application for tracking your daily health and productivity metrics. Track your sleep, work, and mood, visualize trends with interactive charts, and generate weekly PDF reports.  

Built with **Python**, using **PySide6**, **Matplotlib**, **Pandas**, and **ReportLab**.

---

## Features

- Add daily data: Sleep, Work, Mood  
- Interactive trend charts  
- Suggestions based on average metrics  
- Export weekly PDF reports  
- Simple and clean UI  
- CSV persistence (auto-save/load)  
- Fully validated inputs to prevent errors  

---

## Installation

### 1. Clone the repository

```bash
git clone https://github.com/AbolfazlGameMaker/LifeSync.git
cd LifeSync
```

### 2. Install Python libraries

Make sure you have **Python 3.10+** installed. Then run:

```bash
pip install PySide6 matplotlib pandas reportlab
```

### 3. Run the application

```bash
python life_sync.py
```

---

## Usage

1. Launch the application.  
2. Enter **Date (YYYY-MM-DD), Sleep hours, Work hours, and Mood (1–10)**.  
3. Click **Add Data** to save your entry.  
4. View interactive charts with **Show Chart**.  
5. Get personalized suggestions using **Life Sync Suggestions**.  
6. Export weekly PDF reports using **Export Weekly PDF**.
